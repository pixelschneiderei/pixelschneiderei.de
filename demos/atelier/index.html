Design

Eine Form, die den
Inhalten dient.

Jedes Pixel ist gesetzt, weil es etwas erzählt. Kein Stilmittel ohne Funktion, keine Animation ohne Zweck. Die Bühne, nicht das Schauspiel.

Typografie

Eine Schrift,
die Haltung hat.

Variable Fonts mit charakterstarken Italics. Lesbar auf jedem Display, ausdrucksstark in jeder Größe.

Farbe

Tiefe, statt
Buntheit.

Ein präzises Spektrum. Wenig Farbe, viel Wirkung. Damit das Wesentliche im Blick bleibt.

Bewegung

Ruhig. Gezielt. Souverän.

Bewegung folgt Funktion: Sie führt, hebt hervor, bestätigt. Kein Effekt-Karussell, sondern Choreografie. Auf jedem Gerät smooth und ressourcenschonend.

Material

Aus dem
besten Stoff.

Wir verbauen, was Bestand hat. Keine modischen Frameworks ohne Gehalt – sondern bewährte, aktuelle Standards. Schicht für Schicht.

Schicht 1 — Inhalt

Klar strukturiertes HTML, semantisch gehalten. Die Grundlage für SEO und Barrierearmut.

Schicht 2 — Form

Modernes CSS mit Container-Queries, variable Fonts, präzises Grid. Performance und Eleganz im Gleichgewicht.

Schicht 3 — Logik

Schlanke, bewusst eingesetzte JavaScript-Bausteine. Auf Wunsch ein eigenes leichtgewichtiges CMS, Datenbank inklusive Backup.
